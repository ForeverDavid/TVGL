BoundingBox Fields

The BoundingBox type exposes the following members.

Fields

  Name Description
CornerVertices
The corner points
Dimensions
The dimensions of the bounding box. The 3 values correspond to the 3 direction.
Directions
The Directions normal are the three unit vectors that describe the orientation of the box.
PointsOnFaces
The PointsOnFaces is an array of 6 lists which are vertices of the tessellated solid that are on the faces of the bounding box. These are not the corners of the bounding box. They are in the order of direction1-low, direction1-high, direction2-low, direction2-high, direction3-low, direction3-high.
Volume
The volume of the bounding box.
