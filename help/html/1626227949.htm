Methods

  Name Description
Simplify
Simplifies the model by merging the vertices that are closest together
SimplifyByPercentage
Simplifies by the percentage provided. For example, is ts has 100 triangles, then passing a 0.2 will reduce to 80 triangles
SimplifyByTolerance
Simplifies by a tolerance whereby vertices within the specified length will be merged.
