TVGL.Boolean_Operations Namespace

Classes

  Class Description
ContactData
A ContactData object represents a 2D path on the surface of the tesselated solid. It is notably comprised of loops (both positive and negative) and loops are comprised of contact elements).
ContactElement
A ContactElement describes the atomic class for the collective ContactData class which describes how a slice affects the solid. A ContactElement is basically an edge that may or may not correspond to a real edge in the solid. Additionally, it has properties that reference pertinent data in the solid.
Loop
The Loop class is basically a list of ContactElements that form a path. Usually, this path is closed, hence the name "loop", but it may be used and useful for open paths as well.
Slice
The Slice class includes static functions for cutting a tessellated solid.

FaceSplitType
The enumerator, FaceSplitType, is used to describe how a ContactElement within the ContactData relates to the tessellated solid.
